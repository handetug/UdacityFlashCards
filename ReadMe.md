Flash Cards Project
Flash Cards application is the React Native.

This app was developed and tested on both IOS (IPhone 11 PRO(13.3)) and Android (Pixel 2 Andorid 8.2) emulators.

Installing & Deployment
Make sure that you have Node.js

Clone the project and navigate to the cloned folder

Type the command npm install to install all project dependencies

Start the development server with yarn start
